# 🤖 AI Data Analyst

## Overview

AI Data Analyst is an intelligent web application that allows users to upload datasets, perform exploratory data analysis, generate visualizations, train machine learning models, and interact with their data using natural language through open-source AI models.

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Matplotlib

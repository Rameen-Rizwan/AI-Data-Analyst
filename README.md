# 🤖 AI Data Analyst

An intelligent no-code data analytics and machine learning platform built with **Streamlit**. Upload any dataset, perform data analysis, visualize insights, train machine learning models, compare algorithms, generate AI-powered reports, and download trained models—all from a modern interactive dashboard.

## ✨ Features

### 📊 Data Analysis
- Upload CSV and Excel datasets
- Dataset summary and statistics
- Missing value analysis
- Duplicate detection
- Outlier detection
- Correlation heatmap
- Interactive visualizations

### 🤖 AI Insights
- AI-powered dataset summary
- Smart recommendations
- Machine Learning advisor
- Gemini AI assistant
- Business insights generation

### 🧠 Machine Learning Studio
- Automatic target column detection
- Automatic dataset type detection
- One-click dataset cleaning
- Classification and Regression support
- Multiple ML algorithms
- Hyperparameter tuning
- Model comparison
- Cross-validation
- Feature importance visualization
- Performance evaluation
- Model training history

### 📄 Reports & Export
- AI-generated reports
- Download predictions
- Download trained model (.pkl)
- Download cleaned dataset

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/AI-Data-Analyst.git
```

Navigate to the project folder:

```bash
cd AI-Data-Analyst
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
